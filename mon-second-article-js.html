Le Langage JavaScript

Quelques exercices pour comprendre le rôle du JS

Euh... C'est quoi le JavaScript ????

JavaScript est un langage de script orienté objet. Il permet, entre autres, d'introduire sur une page web ou HTML des petites animations, des effets et des interactions entre le site web et son utilisateur.

Après avoir travaillé sur le HTML et le CSS, nous avons travaillé sur des petits exercices de Javascript. Contrairement aux deux autres langages, celui-ci permet de faire des interactions directs avec l'utilisateur.

Par exemple, lui demander son nom, son prénom, son âge, etc, et on peut créer des variables qui récupèrent ces informations et les affichent ensuite de différentes manières.

Variables et conditions

Une variable est un objet repéré par son nom, pouvant contenir des données, qui pourront être modifiées lors de l'exécution du programme. Dans nos exercices, nous avons utilisé les variables pour récupérer les données de champ input, de div ou fenêtre prompt. On utilise var ou let pour définir une variable.

La méthode prompt() affiche une boîte de dialogue, éventuellement avec un message, qui invite l'utilisateur à saisir un texte. On peut voir un exemple de son utilisation sur la capture ci-dessus et sur la vidéo ci-dessous.

 

L'instruction if exécute une instruction si une condition donnée est vraie ou équivalente à vrai. Si la condition n'est pas vérifiée, il est possible d'utiliser une autre instruction : else. 

if (condition) {
                condition1
              } else {
                condition2
              }

Exemples d'exercices

Voici un petit listing de certaines consignes données aux élèves sur les exercices à réaliser :

  • Afficher une boîte de dialogue permettant de saisir son prénom. Afficher dans une boîte de dialogue "Bonjour, " suivi du prénom saisi.

  • Demander à l’utilisateur deux nombres à virgule. Ne garder que la partie entière du premier . Les multiplier et afficher le résultat dans une boîte de dialogue.

  • Demander à l’utilisateur sa pointure et son année de naissance. Créer une fonction qui fait les calculs suivants : Multiplier la pointure par 2, Ajouter 5 au résultat, Multiplier le tout par 50, Soustraire l’année de naissance, Ajouter au tout 1766. La fonction doit retourner le résultat. Tester avec votre date de naissance et votre pointure.

  • Demander à l’utilisateur de saisir son âge. S’il a plus de 18 ans, afficher *" Vous êtes majeur "*. Sinon afficher *" Vous êtes mineur "*.

  • Dans le fichier HTML fourni, changer l'image au survol de celle-ci par la deuxième.

Pour les exercices de niveau 2, un peu plus poussé, les élèves ont appris certains notions de javascript indispensable de type querySelector qui permet de sélectionner un élement HTML par son identifiant ou sa classe, ou addEventListener. Plusieurs événements peuvent-être déclenchés sur une page et il est possible d'écouter ces événements grâce à la méthode addEventListener() et de déclencher une fonction en guise de rétroaction. Les clics (mouseclick) ou les survols (mouseover) sont des exemples d'évènements possibles sur un objet (bouton, image, etc).
