Projet Films

Présentation d'un premier projet mêlant Bootstrap, JavaScript et une API.

Le projet films c'est quoi ?

Le but de cet exercice qui nous a occupé pendant huit séances est d'utiliser une API sur le site de l'Open Movie Database pour créer un catalogue de films à partir du framework Bootstrap que nous avons découvert en début d'année et d'un peu de JavaScript.

C'est quoi une API ? Une API (Application Programming Interface) permet à deux applications de communiquer entre elles. Elle rend disponibles les données ou les fonctionnalités d’une application existante afin que d’autres applications les utilisent.

Les élèves étaient libres de choisir leur maquette pour ce projet. La seule contrainte était d'utiliser le framework Bootstrap et son système de cards pour l'affichage des films. Une card, c'est un bloc de code tout prêt modulable et personnalisable qui permet d'afficher dans une div une image, un titre, du texte et un bouton.

Macéo, Arthur et Timéo ont donc commencé par réaliser un squelette de site intégrant une navbar et des cards de Bootstrap. Pour donner un peu de punch visuel à la page, il a été décidé d'y ajouter un carousel qui ferait défiler 3 ou 4 images de films.

Ensuite, ils ont créé quatre exemples de cards en HTML, chacun avec 4 films de son choix pour comprendre le fonctionnement des cards et leur donner un peu plus de couleur avec du CSS... Ces cards seront au final supprimées et remplacées par d'autres que nous allons créer dynamiquement en JavaScript.

Créer des cards dynamiques en JS

L'étape suivante du projet s'est déroulée sur Codepen. Nous y avons écris le JavaScript servant à récupérer les infos des films.

Les différentes étapes à réaliser étaient :

  • Récupérer les films souhaités par l'API et les enregistrer dans un fichier JSON (le JSON est un format qui stocke des informations structurées et est principalement utilisé pour transférer des données entre un serveur et un client).

  • Afficher les éléments souhaités des films (Poster, titre, année) par JavaScript.

  • Recréer en JavaScript la structure d'une card Bootstrap (Image, title, body, etc).

Intégrer le JS sur notre projet avec VSCode

Une fois le code Javascript fonctionnel et nettoyé de ses erreurs (et il y en a eu !), nous l'avons intégré à notre site créé en tout début de projet avec Bootstrap grâce au logiciel VSCode. Nous avons ensuite corrigé quelques petits bugs d'affichage, liés principalement à des soucis de CSS.

La dernière étape du projet était la mise en ligne du site avec Github, que nous avons appris à utiliser. Il fallait d'abord installer git sur les ordinateurs, se connecter à son compte et créer le repository (dossier) qui allait contenir le projet en ligne, préparer l'envoi, ajouter les fichiers à la liste d'envoi et ensuite envoyer (push avec Github). Nous avons utilisé l'invite de commande Windows pour les différentes étapes de cette partie qui se déroulait entièrement en lignes de commande.

Les projets des élèves sont enfin en ligne. Vous pouvez les consulter ici et ici.

Ce blog est entièrement conçu avec le système de grille de Bootstrap.
